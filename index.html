Over Mij

Studieachtergrond

Ik heb voor 4 jaar lang op Oud-Erasmus gezeten,
zo begon ik eerst met Mavo/Havo niveau in het eerste jaar
en heb ik voor de laatste 3 jaar op mavo niveau verder
gevolgd en heb ik mijn mavo diploma gekregen.

Over Mezelf

Ik ben Marc Stokhof, 17 jaar oud, en ik woon in Almelo.
Mijn hobby's zijn gamen en voetbal, wat ik als sport doe.
Daarnaast ben ik heel gedreven, leer ik snel en zet ik me
altijd volledig in om beter te worden in wat ik doe.

Waarom Software Developer

Ik heb voor de opleiding Software Developer gekozen
omdat ik het ontwerpen en bouwen van software erg interessant vind.
Ik wil graag meer leren over programmeren
en daarom steek ik hier met plezier mijn energie in.

Mijn Werk

Webwinkel maken

Ik heb een webwinkel gebouwd waarin je net als bij bol.com
producten kunt kopen. Je kunt artikelen bekijken,
toevoegen aan je winkelwagen en een bestelling plaatsen.

Ook heb ik gewerkt aan een overzichtelijke indeling zodat gebruikers
makkelijk kunnen navigeren.

Chat app

Ik heb een chat app gemaakt waarmee gebruikers berichten kunnen sturen.
Je kunt ook afbeeldingen versturen en alles wordt opgeslagen in een database.
Daarnaast heb ik gezorgd dat de chat in realtime werken.

Pizza Web

Ik heb een HTML-website gemaakt over pizza's.
Daarop kun je verschillende pizza’s bekijken en informatie vinden over
de ingrediënten. De website heeft een simpele en duidelijke layout zodat
alles makkelijk te vinden is.

Technologieën/Vaardigheden

Tot nu toe heb ik het meeste gewerkt met deze 3 technologieën

CSS


Met CSS heb ik al een aantal opdrachten gemaakt.
Zo heb dit leren kennen door het vak Webdesign en
met behulp van W3Schools en AI heb ik steeds meer
nieuwe dingen ontdekt. In de toekomst zou ik
CSS gebruiken voor het opmaken van websites.

HTML


Met HTML heb ik het meeste gewerkt voor opdrachten.
Zo heb dit leren kennen door het vak Webdesign
en ook door gebruik van W3Schools. Voor in de toekomst
denk ik dat ik het zou gebruiken voor het maken van
verschillende websites

C#


Met C# heb ik veel projecten gemaakt.
C# heb ik leren kennen door het vak Programmeren
en zo heb ik ook veel gebruik gemaakt van W3Schools.
Voor in de toekomst zou ik dit gebruiken voor het
bouwen van mobiele apps bijvoorbeeld.

Side/toekomstige projecten

Ik zou 2 verschillende projecten willen maken voor de toekomst

Website maken


Ik zou een website willen maken waar
mensen met elkaar kunnen praten en spelletjes kunnen doen.
Een beetje zoals Omegle, waar gebruikers met elkaar kunnen
communiceren en samen plezier kunnen hebben.

Mobiele App bouwen


Ik wil graag een mobiele app bouwen, specifiek een
taaluitwisselingsapp. Deze app verbindt mensen wereldwijd
om elkaar te helpen bij het leren van talen. Gebruikers kunnen
chatten, spraakberichten uitwisselen, videobellen en, net als
bij Duolingo, nieuwe talen leren.
